CRM Video Sending Module

The CRM Video Sending module is a powerful tool that revolutionizes the way you connect with your customers. It allows you to send personalized videos and promotional text messages, providing a unique and engaging customer experience.

Features

  • Data Upload: Upload company information manually or by importing a CSV or Excel file. This flexibility allows for easy and efficient data management.
  • Personalized Videos: Generate personalized videos using the HeyGen service. These videos can be tailored to each recipient, making your communications more personal and effective.
  • Email Integration: Send the personalized videos directly to your customers' inbox. This feature ensures that your communications reach your customers in a timely manner.
  • Text Promotions: In addition to videos, you can also send promotional text messages. This feature provides an alternative way to reach out to your customers.

API Key Setup

To ensure the proper functioning of the CRM Video Sending module, you must set up the HEYGEN_API_KEY and OPENAI_API_KEY as system environment variables. These keys are essential for the module to interact with the HeyGen service and OpenAI respectively.

Python Dependencies

The CRM Video Sending module requires the following Python libraries: openai, requests, unidecode, googlesearch-python, and openpyxl. Please ensure these are installed in your Python environment for the module to function correctly.
